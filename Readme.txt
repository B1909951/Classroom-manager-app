Hướng dẫn thêm dữ liệu từ các collection
1. Tạo database tên classroom-manager-app
2. Lần lượt tạo các collection tên teachers, classrooms, students
3. Lần lượt thực hiện add data bằng các file teachers.json, classrooms.json, students.json 

Hướng dẫn chạy dự án
1. Mở terminal tại classroom_manager-backend và chạy lệnh: npm start
2. Mở terminal tại classroom_manager-frontend và chạy lệnh: npm run dev'
3. Mở trình duyệt và truy cập vào địa chỉ http://127.0.0.1:3001/ để truy cập vào app
